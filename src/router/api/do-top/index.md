## ------------- 问答、资源文件、小说、博客文章置顶操作 ----------------

#### 简要描述

- `pc | web | app | wechat` 端
- 问答、资源文件、小说、博客文章置顶操作
- 只有超级管理员角色可操作

#### 请求

- `get | post`
- `top/add`

#### 参数

| 参数名 | 类型 | 是否必填 | 说明 |
|:---:|:---:|:---:|:---:|
| id | string | 是 | 置顶资源的id |
| type | string | 是 | 置顶资源类型，现可取 502 503 504 505 |

#### 返回示例

```
{
  "code": 200,
  "message": "操作成功",
  "data": null,
  "total": 0
}
```
