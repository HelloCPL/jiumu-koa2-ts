
## ---------------- 新增用户自定义标签 ---------------------

#### 简要描述

- `pc | web | app | wechat` 端
- 新增用户自定义标签

#### 请求

- `get | post` 
- `tag/custom/add`

#### 参数

| 参数名 | 类型 | 是否必填 | 说明 |
|:---:|:---:|:---:|:---:|
| label | string | 是 | 用户自定义标签说明 |
| sort | mediumint | 否 | 排序，值越小越前，默认1 |

#### 返回示例

```
{
  "code": 200,
  "message": "操作成功",
  "data": null,
  "total": 0
}
```

## ---------------- 获取指定用户自定义标签 ---------------------

#### 简要描述

- `pc | web | app | wechat` 端
- 获取指定用户自定义标签，仅可获取自己创建的自定义标签
- 返回数组或[]

#### 请求

- `get | post` 
- `tag/custom/get/byids`

#### 参数

| 参数名 | 类型 | 是否必填 | 说明 |
|:---:|:---:|:---:|:---:|
| ids | string | 是 | 用户自定义标签 id 集合，多个用逗号隔开 |

#### 返回示例

```
{
  "code": 200,
  "message": "操作成功",
  "data": [
    {
      "id": "c3221e5e-bb1a-4220-b178-26d37fa1ade0",
      "label": "java",
      "sort": 1,
      "createTime": "2021-08-18 03:12:05",
      "updateTime": "2021-08-18 03:12:05",
      "terminal": "管理端"
    }
  ],
  "total": 0
}
```

## ---------------- 获取用户自定义标签列表 ---------------------

#### 简要描述

- `pc | web | app | wechat` 端
- 获取用户自定义标签列表，仅可获取自己创建的自定义标签
- 返回数组或[]

#### 请求

- `get | post` 
- `tag/custom/get/byids`

#### 参数

| 参数名 | 类型 | 是否必填 | 说明 |
|:---:|:---:|:---:|:---:|
| pageNo | number | 否 | 页码，默认 1 |
| pageSize | number | 否 | 每页页数，默认 10 |

#### 返回示例

```
{
  "code": 200,
  "message": "操作成功",
  "data": [
    {
      "id": "329573a2-fd89-4fbf-aef6-3a0b7bc5abc3",
      "label": "vue",
      "sort": 1,
      "createTime": "2021-08-18 03:11:20",
      "updateTime": "2021-08-18 03:11:20",
      "terminal": "管理端"
    }
  ],
  "total": 4
}
```
